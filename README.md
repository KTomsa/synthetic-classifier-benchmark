# Simulation Conference Project

This project is developed for academic research to systematically compare various classification algorithms utilizing simulated datasets. 

Note that some names and comments are in Czech language.

## Features
- Generates simulated datasets with varying dimensions, sizes, and correlation structures.
- Trains standard classification algorithms including Random Forest, XGBoost, Gaussian naive Bayes, neural network and Logistic Regression.
- Evaluates model performance using metrics such as accuracy, precision, recall, F1-score, and AUC-ROC.

## Setup
Install the necessary dependencies from `requirements.txt`:
```bash
pip install -r requirements.txt
```

## Usage
Run `main.py` to start the simulation experiments:
```bash
python main.py
```
